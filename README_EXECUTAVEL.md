# Sistema de Controle de Estoque - Executável

## 📦 Conteúdo da Distribuição

Este pacote contém todos os arquivos necessários para executar o Sistema de Controle de Estoque:

### Arquivos Principais
- **SistemaEstoque.exe** - Executável principal do sistema (52MB)
- **estoque.db** - Banco de dados SQLite com as tabelas do sistema
- **Imagem de não-encontado.jpg** - Imagem padrão para produtos sem foto
- **setup_completo.py** - Script de configuração inicial (opcional)

## 🚀 Como Usar

### Instalação Simples
1. Copie toda a pasta `dist` para o local desejado
2. Execute `SistemaEstoque.exe`
3. O sistema iniciará automaticamente

### Primeira Execução
- O sistema criará automaticamente as tabelas necessárias
- Todas as funcionalidades estarão disponíveis imediatamente
- O banco de dados já contém a estrutura completa

## 🔧 Requisitos do Sistema

- **Sistema Operacional**: Windows 7 ou superior
- **Memória RAM**: Mínimo 2GB
- **Espaço em Disco**: 100MB livres
- **Resolução**: Mínimo 1024x768

## 📋 Notas Importantes

### Banco de Dados
- O arquivo `estoque.db` contém todos os dados
- **IMPORTANTE**: Sempre faça backup deste arquivo
- Não delete ou mova este arquivo da pasta

### Atualizações
- Para atualizar, substitua apenas o `SistemaEstoque.exe`
- Mantenha sempre o `estoque.db` original
- Suas configurações e dados serão preservados

### Suporte
- Sistema desenvolvido com Python + Tkinter
- Interface gráfica nativa do Windows
- Banco de dados SQLite integrado
