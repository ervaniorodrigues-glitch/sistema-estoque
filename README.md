# Sistema de Controle de Estoque

Sistema web completo para gerenciamento de estoque, produtos, fornecedores, clientes e movimentações.

## Funcionalidades

- ✅ Cadastro de Produtos com fotos
- ✅ Controle de Estoque (Entradas/Saídas)
- ✅ Gestão de Fornecedores e Clientes
- ✅ Controle de Funcionários
- ✅ Sistema de Empréstimos
- ✅ Relatórios de Movimentações
- ✅ Exportação para Excel e PDF
- ✅ Backup automático do banco de dados
- ✅ Sistema de usuários com autenticação

## Tecnologias

- Python 3.11
- Flask 3.0
- SQLite
- Bootstrap 5
- Font Awesome

## Deploy no Render

1. Crie uma conta no [Render](https://render.com)
2. Conecte seu repositório GitHub
3. O Render detectará automaticamente o `render.yaml`
4. Clique em "Deploy"

## Executar Localmente

```bash
pip install -r requirements.txt
python estoque_web/app.py
```

Acesse: http://127.0.0.1:5000
